# Руководство администратора ReviewHub

## Начало работы

### Регистрация

1. Откройте `https://admin.reviewhub.ru/register`
2. Заполните:
   - **Email** — рабочий email
   - **Пароль** — минимум 8 символов
   - **Название компании** — отображается клиентам в форме отзыва
   - **Телефон** — формат +7XXXXXXXXXX
3. Нажмите «Зарегистрироваться»

### Вход

1. Откройте `https://admin.reviewhub.ru/login`
2. Введите email и пароль
3. Сессия действует 7 дней (автоматическое обновление токена)

## Настройка профиля компании

**Страница:** Настройки (`/settings`)

| Поле | Описание | Обязательное |
|------|----------|:------------:|
| Название компании | Показывается клиентам в PWA | Да |
| Ссылка Яндекс Карты | URL вашей организации на Яндекс Картах | Да |
| Размер скидки (%) | Скидка для негативных отзывов | Нет (по умолч. 10%) |
| Текст скидки | Описание промо-предложения | Нет |

### Как найти ссылку на Яндекс Карты

1. Откройте [Яндекс Карты](https://yandex.ru/maps)
2. Найдите свою организацию
3. Скопируйте URL из адресной строки
4. Формат: `https://yandex.ru/maps/org/название/123456789/`

## Управление клиентами

**Страница:** Клиенты (`/clients`)

### Добавление клиента вручную

1. Нажмите «Добавить клиента»
2. Заполните имя и телефон (+7XXXXXXXXXX)
3. Email — необязательно

### Импорт из CSV

1. Подготовьте CSV-файл (максимум 10 МБ):
   ```csv
   name,phone,email
   Иван Петров,+79001234567,ivan@mail.ru
   Мария Сидорова,+79009876543,
   ```
2. Нажмите «Импорт CSV»
3. Выберите файл
4. Система покажет: импортировано / пропущено / ошибки

### Удаление клиента

Нажмите иконку удаления рядом с клиентом. Удаление необратимо — все связанные запросы и отзывы также удаляются.

## Настройка мессенджеров

**Страница:** Настройки (`/settings`)

ReviewHub поддерживает отправку через SMS, Telegram и Max. Для настройки мессенджеров:

### Telegram

1. Создайте бота через [@BotFather](https://t.me/BotFather) в Telegram
2. Скопируйте токен бота
3. Вставьте токен в поле «Telegram Bot Token» на странице настроек
4. Система автоматически проверит токен и зарегистрирует вебхук
5. Имя бота отобразится после успешной настройки

> **Примечание (self-signed SSL):** При использовании самоподписанного сертификата система автоматически загружает его при регистрации вебхука в Telegram API. Это необходимо для приёма обновлений от Telegram на HTTPS-эндпоинт без сертификата от CA.

### Max (max.ru)

1. Создайте бота в [Max](https://max.ru) через настройки
2. Скопируйте токен бота
3. Вставьте токен в поле «Max Bot Token» на странице настроек
4. Система автоматически проверит токен и зарегистрирует вебхук

### Автоматическое подключение клиентов

Клиенты автоматически подключаются к мессенджерам через deep links:
- В SMS-сообщении клиент получает ссылку на бота
- При переходе по ссылке клиент автоматически привязывается к мессенджеру
- Следующие уведомления приходят через мессенджер (бесплатно!)

## Отправка запросов на отзыв

### Разовая отправка

1. На странице «Клиенты» найдите нужного клиента
2. Выберите канал отправки (SMS / Telegram / Max)
3. Нажмите кнопку отправки
4. Клиент получит сообщение со ссылкой на PWA

### Массовая отправка

1. На странице «Клиенты» отметьте нужных клиентов чекбоксами
2. Используйте «Выбрать все» для выделения всех активных клиентов
3. Выберите канал отправки
4. Нажмите «Отправить» в панели действий
5. Система покажет результат: отправлено / ошибок

### Что получает клиент

SMS-сообщение:
```
{Название компании} просит оставить отзыв: https://review.reviewhub.ru/review/{token}

Отписка: https://review.reviewhub.ru/optout/{token}

📲 Получайте уведомления без SMS — подпишитесь на наш бот:
Telegram: t.me/{bot}?start={id}
Max: max.ru/{bot}?start={id}
```

Ссылки на мессенджеры добавляются только если бот настроен и клиент ещё не подключён к мессенджеру. В мессенджерах клиент получает сообщение с форматированным текстом и ссылками.

### Fallback на SMS

Если отправка через мессенджер не удалась (бот заблокирован, ошибка API), система автоматически отправит SMS.

## Просмотр отзывов

**Страница:** Отзывы (`/reviews`)

Каждый отзыв содержит:
- Рейтинг (1-5 звёзд)
- Текст отзыва
- Тональность (положительный / отрицательный / нейтральный)
- Маршрут (Яндекс Карты / скрытый)
- Промо-код (если негативный)

### Фильтры
- Все отзывы
- Только положительные
- Только отрицательные

## Аналитика

**Страница:** Дашборд (`/`)

### Метрики

| Метрика | Описание |
|---------|----------|
| Сообщений отправлено | SMS + мессенджеры за период |
| Отзывов | Количество полученных отзывов |
| Конверсия | Отзывы / сообщения (%) |
| Средний рейтинг | Средний балл (1-5) |
| Положительные | Количество положительных отзывов |
| Отрицательные | Количество отрицательных отзывов |

### Аналитика по каналам

Таблица «По каналам» показывает для каждого канала (SMS, Telegram, Max):
- **Отправлено** — количество успешных отправок
- **Ошибки** — количество неудачных попыток
- **Отзывы** — количество полученных отзывов через этот канал
- **Конверсия** — процент отзывов от отправленных
- **Fallback на SMS** — сколько раз мессенджер не сработал и система переключилась на SMS

### Периоды
- 7 дней
- 30 дней (по умолчанию)
- 90 дней

### График
Количество отзывов по дням за выбранный период.

## Маршрутизация отзывов

Система автоматически направляет отзывы:

| Тональность | Уверенность | Звёзды | Действие |
|-------------|-------------|--------|----------|
| Положительная | >= 70% | Любые | Redirect на Яндекс Карты |
| Положительная | < 70% | >= 4 | Redirect на Яндекс Карты |
| Положительная | < 70% | < 4 | Скрытый раздел |
| Отрицательная | Любая | Любые | Скрытый раздел + промо-код |

## Отписка клиентов

Если клиент нажимает ссылку «Отписка» в SMS:
- Клиент помечается как отписавшийся
- Все запланированные напоминания отменяются
- Новые SMS этому клиенту не отправляются

## Безопасность

- Все телефоны клиентов зашифрованы (AES-256-GCM)
- Данные хранятся на российском сервере (152-ФЗ)
- Пароли хешированы (bcrypt)
- Сессия автоматически завершается через 7 дней

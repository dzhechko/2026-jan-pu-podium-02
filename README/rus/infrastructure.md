## Docker-сервисы

```
┌──────────────────────────────────────────────────────────┐
│              NGINX (порты 80, 443, 9443)                 │
│         SSL (самоподписанный) + Reverse Proxy            │
├──────────────────────────────┬───────────────────────────┤
│  :443 — Admin (статика) +   │  :9443 — PWA (статика) +  │
│         API (/api/)          │          API (/api/)       │
├──────────────────────────────┴─────┬─────────────────────┤
│                                    │                     │
│         PostgreSQL 16              │     Redis 7         │
│         (внутренний)               │   (внутренний)      │
│         1 ГБ RAM                   │    128 МБ RAM       │
└────────────────────────────────────┴─────────────────────┘
```

### Распределение ресурсов

| Сервис | RAM | CPU | Диск |
|--------|-----|-----|------|
| API (Node.js) | 512 МБ | 1 vCPU | — |
| PostgreSQL | 1 ГБ | 1 vCPU | ~50 ГБ |
| Redis | 128 МБ | 0.5 vCPU | — |
| Nginx | 64 МБ | 0.5 vCPU | — |
| Admin (static) | 64 МБ | — | ~10 МБ |
| PWA (static) | 64 МБ | — | ~5 МБ |
| ОС + Docker | 2 ГБ | 1 vCPU | ~10 ГБ |
| **Итого** | **~4 ГБ** | **4 vCPU** | **~65 ГБ** |

## Сетевые требования

### Внешние подключения (исходящие)

| Сервис | URL | Порт | Назначение |
|--------|-----|------|------------|
| SMSC.ru | `smsc.ru/sys/send.php` | 443 | SMS-рассылка |
| Anthropic | `api.anthropic.com` | 443 | Анализ тональности |
| GitHub | `github.com` | 443 | CI/CD деплой |

### Входящие порты

| Порт | Сервис | Назначение |
|------|--------|------------|
| 80 | Nginx | HTTP -> HTTPS redirect |
| 443 | Nginx | HTTPS (админ-панель + API) |
| 9443 | Nginx | HTTPS (PWA форма отзывов) |
| 22 | SSH | Администрирование |

## SSL-сертификаты

### Production (текущая конфигурация)
- **Тип:** Самоподписанный сертификат
- **CN/SAN:** IP `89.125.130.105`
- **Срок:** 365 дней (ручное обновление)
- **Расположение:** `nginx/ssl/selfsigned.crt`, `nginx/ssl/selfsigned.key`
- **Telegram:** сертификат загружается при регистрации вебхука

### При наличии домена
- **Провайдер:** Let's Encrypt (бесплатно)
- **Обновление:** автоматическое через certbot
- **Домены:** `admin.reviewhub.ru`, `review.reviewhub.ru`

## Резервное копирование

- PostgreSQL: `pg_dump` ежедневно в 03:00 МСК
- Хранение: 30 ежедневных + 12 еженедельных
- Тест восстановления: ежемесячно
- Процедура восстановления: см. [Развертывание](deployment.md#восстановление-из-бэкапа)

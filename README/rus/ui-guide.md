# Интерфейс ReviewHub

## Админ-панель

### Навигация

Боковое меню (sidebar) с разделами:
- **Дашборд** — главная страница с аналитикой
- **Клиенты** — управление базой клиентов
- **Отзывы** — просмотр полученных отзывов
- **Настройки** — профиль компании
- **Выйти** — завершение сессии

## Дизайн-система

- **CSS-фреймворк:** Tailwind CSS
- **Компоненты:** Shadcn/ui (админ-панель)
- **Иконки:** Lucide React
- **Шрифт:** системный стек (sans-serif)
- **Цвета:** светлая тема, акценты через Tailwind
- **Адаптивность:** мобильный-first для PWA, десктоп-first для админ-панели

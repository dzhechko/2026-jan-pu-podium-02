# Product Discovery Brief: Podium Clone (Russia)
**Mode:** QUICK | **Date:** 2026-03-11

## A. ONE-LINER

> **ReviewHub (Подиум-аналог)** — это Podium + русская локализация (Яндекс Карты, SMSC.ru, 152-ФЗ) для автоматического сбора отзывов через SMS с интеллектуальной маршрутизацией по тональности.

## B. PROBLEM STATEMENT (Before / After)

| Измерение | ❌ Без ReviewHub | ✅ С ReviewHub | Улучшение |
|-----------|-----------------|----------------|-----------|
| **Время** | Ручной обзвон/просьбы об отзывах | Автоматическая SMS-рассылка | 10x быстрее |
| **Деньги** | Сборная солянка: AmoCRM + Wazzup24 + RocketData = ₽20-30K/мес | Единая платформа ₽3-10K/мес | 3-5x дешевле |
| **Усилия** | Ручная модерация, копирование отзывов | LLM-анализ тональности + авто-маршрутизация | Автоматизация |
| **Результат** | 2-5 отзывов в месяц | 15-30 отзывов в месяц | 5-10x больше отзывов |

## D. SIMPLIFIED CJM (User-Specified)

```
1. Админ → Отправляет SMS через SMSC.ru (авто после услуги или вручную)
   ↓
2. Клиент → Получает SMS со ссылкой на PWA
   ↓
3. Клиент → Открывает PWA, оставляет отзыв (текст + оценка)
   ↓
4. Система → LLM анализирует тональность отзыва
   ↓
5a. Позитивный → Перенаправление на Яндекс Карты (клиент сам публикует)
5b. Негативный → Сохраняется в скрытом разделе сайта компании
   ↓
6. Если отзыв не оставлен → Серия напоминающих SMS (психологический алгоритм)
```

### Ключевое ограничение: Яндекс Карты
**КРИТИЧЕСКИ ВАЖНО:** Яндекс Карты **НЕ ИМЕЕТ публичного API для публикации отзывов**. Поэтому:
- Позитивные отзывы: система перенаправляет клиента по ссылке на страницу бизнеса на Яндекс Картах, где клиент сам оставляет отзыв
- Это стандартный подход Podium: redirect happy customers → review platform
- Ответы на отзывы Яндекс Карт: только через Яндекс Бизнес кабинет (нет API)
- Модерация отзывов Яндексом: автоматическая + ручная, до 2 недель

### SMS-провайдер: SMSC.ru
- 25+ лет на рынке, 92% доставляемость
- Поддержка: SMS, Viber, Email, голосовые вызовы
- API для интеграции: https://www.smsc.ru/api/
- Стоимость: от ~2 руб/SMS

### Алгоритм напоминаний (психологически обоснованный)
Базируется на принципах behavioral nudging:
1. **Через 2 часа** после услуги — первое SMS (свежие впечатления)
2. **Через 24 часа** — напоминание с другой формулировкой
3. **Через 3 дня** — упоминание скидки за отзыв
4. **Через 7 дней** — финальное напоминание с усиленным оффером
5. **Стоп** — после 4 попыток или после оставления отзыва
- Используется принцип reciprocity (скидка за отзыв) + scarcity (ограниченное предложение)
- Каждое SMS — уникальный текст, чтобы не вызывать раздражение
